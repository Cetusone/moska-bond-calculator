# Moska Bond Calculator

**Moska Bond Calculator** — это backend-микросервис для расчета реальной доходности облигаций. Приложение агрегирует рыночные данные из нескольких источников (Московская биржа, Т-Банк Инвестиции) и предоставляет детализированный расчет доходности с учетом налогов, валютной переоценки и накопленного купонного дохода (НКД).

## Ключевые возможности

- **Продвинутый калькулятор доходности:** Расчет полной и чистой доходности в рублях, расчет эффективности бумаги в валюте номинала.
- **Учет налогов (НДФЛ):** Автоматическое вычисление налога (13%) на прибыль от курсовой разницы и купонов.
- **Влияние валютного курса:** Оценка `currency effect` для облигаций, номинированных в иностранной валюте.
- **Интеграция с MOEX ISS API:** Получение актуальных котировок, НКД и параметров бумаги напрямую с Московской биржи.
- **Интеграция с Tinkoff Invest API:** Использование gRPC/REST (PIAPI) для загрузки купонных расписаний и параметров инструментов.
- **Service Discovery:** Поддержка обнаружения сервисов (Eureka/Consul) через `@EnableDiscoveryClient`.

## Технологический стек

- **Язык:** Java 17+ (активное использование `record` и современных языковых конструкций)
- **Фреймворк:** Spring Boot 4.0.3
- **Интеграции:** - Spring Web (`RestClient` для MOEX)
  - Tinkoff PIAPI (`ru.tinkoff.piapi.core.InvestApi`)
- **Архитектура:** REST API, Service Layer, Strategy/Adapter (для работы с разными источниками данных).
